# :material-play-circle: Mediaplayer

!!! abstract "Overview"
    * **Access:** `Editor` & `Reader`
    * **Editor:** Edit metadata, correct transcripts, and fix translations.
    * **Reader:** View media and jump to specific timestamps by clicking transcript text.
