# :material-magnify: Searchpage

## Overview

!!! abstract "Overview"
    * **Access:** `Editor` & `Reader`
    * **Function:** Filter speaker statements and search in them
    * [**Speaker Statements:**](#speaker-statements) :material-chat-outline: The search engine contains full statements not just subtitles.
    * **Filtering:** Filter by metadata criteria, that are added at the mediaplayer page by an editor
    * **Search:** Fulltext search by term in the speaker statements

![Searchpage](../static/content/userguide/searchpage/searchpage.png "Search interface to make speaker statements searchable"){ width="600" }

## Speaker Statements

:material-chat-outline: For the search engine the subtitles have been aggregated into speaker statements, that contain the full statement that a speaker made in his turn in a debate. These speaker statements also contain metadata about the speaker and the debate.
