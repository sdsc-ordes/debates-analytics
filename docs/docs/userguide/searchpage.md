# :material-magnify: Searchpage

## Overview

!!! abstract "Overview"
    * **Access:** `Editor` & `Reader`
    * **Function:** Filter speaker statements and search in them
    * [**Speaker Statements:**](#speaker-statements) :material-chat-outline: The search engine contains full statements not just subtitles.
    * **Filtering:** Filter by metadata criteria, that are added at the mediaplayer page by an editor
    * **Search:** Fulltext search by term in the speaker statements

![Searchpage](../static/content/userguide/searchpage/searchpage.png "Search interface to make speaker statements searchable"){ width="100%" }

## Speaker Statements

:material-chat-outline: For the search engine the subtitles have been aggregated into speaker statements, that contain the full statement that a speaker made in his turn in a debate. These speaker statements also contain metadata about the speaker and the debate.

![Searchpage](../static/content/userguide/searchpage/speaker-statement-before-metadata.png "Speaker statement before metadata has been added"){ width="100%" }

![Searchpage](../static/content/userguide/searchpage/speaker-statement-after-metadata.png "Speaker statement after metadata has been added"){ width="100%" }

## Search by term and Filtering

* Search results can be filtered by clicking on tags.
* Additionally a search term can be entered into the search box. The search term is then marked in the search results.

![Searchpage](../static/content/userguide/searchpage/filtering-searchresults.png "Filtering and search by term"){ width="100%" }
